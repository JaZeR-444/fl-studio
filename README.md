# FL Studio Reference Application

Comprehensive reference hub for FL Studio with workflows, tutorials, and music production guides.

## Features
- FL Studio workflow diagrams
- Software tutorials and guides
- Music production techniques
- Feature comparison tables

## GitHub Page
Explore the GitHub Page for this project [here](https://JaZeR-444.github.io/fl-studio/)!

## Getting Started

```bash
pnpm dev
```

This will start the application on port 3003.
